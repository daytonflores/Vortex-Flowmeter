Calculates truncated (integer portion) square root of an integer in the range \mbox{[}0, (2$^\wedge$31) -\/ 1\mbox{]}. 

This will be done using the Bisection Method (\href{https://en.wikipedia.org/wiki/Bisection_method}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Bisection\+\_\+method}}) with range \mbox{[}a, b\mbox{]} and midpoint c. Initial values\+: a = 0 (smallest possible square root) b = 256 (largest possible square root) c = 128 The function will recursively calculate whether c is the exact root of input x. If so, then finish. Else if c$^\wedge$2 \texorpdfstring{$<$}{<} x, then set lower bound a to c. Else if c$^\wedge$2 \texorpdfstring{$>$}{>} x, then set upper bound b to c. The midpoint c will be recalculated within the new range until either\+: c is the exact square root of x \char`\"{}new\char`\"{} c == \char`\"{}old\char`\"{} c (meaning integer approximation is within 1)
