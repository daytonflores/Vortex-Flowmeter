double get_Freq ( uint16_t *  samples)

  • calculates frequency from 1000 samples using peak detection
    the difference between 2 samples is evaluated for +/- positive = moving down the curve, i.e. sample1 is > sample2 negative = moving up the curve, i.e. sample1 is < sample2 when a difference in dir is found, we have crossed a peak / trough when the peak is found, the index (sample #) is stored difference between the two peaks sample # is the taken as the period divide period by sample rate to get signal frequency
